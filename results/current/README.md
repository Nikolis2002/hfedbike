# results/current/

Per-node federated run outputs. 16 files named `<NODE>_results.csv`
(A, B, C, D, E, F, G, H, L, P, Q, R, W, X, Y, Z), each with one header
row + 52 week rows. All CSVs are gitignored; this directory is populated
by `docker compose up` via the mount declared in `docker-compose.yml`.

See `../README.md` for schema and how to regenerate.
